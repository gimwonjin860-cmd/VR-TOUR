# SKP VR Tour Editor

360° VR 파노라마 투어를 만들고 관리하는 웹 에디터입니다.

## 파일 구조

```
index.html           → VR 투어 목록 페이지
vr-tour-editor.html  → 360° VR 투어 에디터
```

## 기능

- 360° equirectangular 이미지 업로드 (드래그 & 드롭)
- 핫스팟 추가/편집 (장면 이동, 정보 팝업)
- 장면 간 연결 설정
- 독립 실행 HTML 파일로 내보내기
- SKP 디자인 시스템 적용

## GitHub Pages 배포

1. 이 레포의 **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: `main` / `/ (root)` 선택 → **Save**
4. 몇 분 후 `https://[username].github.io/[repo-name]/` 에서 확인

## 권장 이미지 해상도

| 등급 | 해상도 | 화질 |
|------|--------|------|
| 최소 | 4096×2048 | 보통 |
| 권장 | 8192×4096 | 좋음 |
| 최고 | 16384×8192 | 최상 |
